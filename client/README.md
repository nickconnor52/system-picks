# The System

> There's always an edge, we're here to find it

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run dist/index (Prod Static Filtes)
node server.js
```
